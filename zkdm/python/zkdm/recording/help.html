录像服务接口定义

一、API格式及说明

1、Http Method

   Get方式

2、返回的接送数据格式如下

   {

      “result”:result,

      “info”:info,

   }

   标黄部分说明：

   给录播机命令发送成功result为“ok”，发送失败result为“error”，注意result只代表是 否把命令发送给了录播机，并不代表命令执行结果。

3、注意，标*的表示arm版本录播机不支持

二、命令

1）录像开始：RecordCmd=StartRecord

2）录像暂停：RecordCmd=PauseRecord

3）停止录像：RecordCmd=StopRecord

4）恢复录像：RecordCmd=ResumeRecord

5）开始RTMP直播：RecordCmd=RTMPLiving

6）结束直播：BroadCastCmd=StopBroadCast

7）获取RTSP流地址：RecordCmd=RtspPreview

8）录像服务状态获取：RecordCmd=QueryRAllInfo

9）*录像文件总路径设置：   RecordCmd=SetFileProperty&FileFormat=mp4&TotalFilePath=D:/rec

10）录像文件子路径设置：

RecordCmd=SetFileFolder&SubFileFolder=指定路径

11) *图片作为片头：

   设置：RecordCmd=OpenPicFile&FilePath={0}&Key=3100&StartTime=0

   开始：RecordCmd=PlayPicFile&Key=3100

   停止：RecordCmd=StopPicFile&Key=3100

12) *视频作为片头：

   设置：RecordCmd=OpenVideoFile&FilePath={0}&Key=3100&StartTime=0

   开始：RecordCmd=PlayVideoFile&Key=3100

   停止：RecordCmd=ChangeCard&CardPort=3100&isMain=true

13) 录播模式：RecordCmd=SetRecordMode&RecordMode=All/Resource/Movie

14) 课程信息设置：

RecordCmd=SetCourseInfo&Department={0}&Subject={1}&CourseName={2}

&Teacher={3}&Address={4}&Grade={5}&DateTime={6}&Description={7}

15) *是否进行某路资源的录播：

   *1：录制：RecordCmd=SetResRecordEnable&CardPort={0}&IsRecord=true

   *2：不录制：RecordCmd=SetResRecordEnable&CardPort={0}&IsRecord=false

16) *设置每一路视频的声音：

RecordCmd=SetVolume&CardPort={0}&Volume={1}

17) *电影录像参数设置：

    RecordCmd=SetFilmParams& BitRate={0}&RecordFormat={1}

    &FrameRate={2}&Width={3}&Height={4}

18) *每路资源的录像参数设置:

    RecordCmd=SetResParams& CardPort={0}&BitRate={1}

    &RecordFormat={2}

19) *设置叠画持续时间：RecordCmd=SetDisSolve&DissolveDuraTime={0}

20) *每路源基本信息设置：

    RecordCmd=SetCardInfo&CardPort={0}&IsUse={1}&IsLock={2}

    &CardName={3}

21）*切换窗口：

RecordCmd =ChangeCard&CardPort=3000&isMain=true

22）*窗口添加特效：

   RecordCmd=SetCardEffect&CardPort={0}&IsOpen=true&EffectType={1}

    其中EffectType为0-10，

{ "FadeInOut", "0" },淡入淡出

  { "LtoR", "1" }, 由左到右

{ "RtoL", "2" }, 由右到左

{ "UtoD", "3" }, 由上到下

{ "DtoU", "4" }, 由下到上

{ "LUtoRD", "5" }, 由左上到右下

  { "RUtoLD", "6" }, 有右上到左下

{ "LDtoRU", "7" }, 由左下到右上

{ "RDtoLU", "8" }, 由右下到左上

{"WindowblindH","9"}, 水平百叶窗

{"WindowblindC","10"}, 垂直百叶窗

23）*没有特效：RecordCmd=SetCardEffect&CardPort={0}&IsOpen=false

24）*全部命令设置：RecordCmd=SetAllEffect&EffectType={0}

25）*画中画控制协议：

*1.子画面选取： RecordCmd=SetPicInPicCardPort&CardPort=3000&isMain=true

*2.模式选取：

RecordCmd=SetPicInPicMode&PicInPicMode=LeftRight/Tile&isMain=true

如果模式为Insert模式：    RecordCmd=SetPicInPicMode&PicInPicMode={0}&Pos={1},{2}&Size={3},{ 4}&CurrPos=LU/LD/RU/RD&isMain={5}

如果模式为Pnp模式（全景）：

RecordCmd=SetPnp&CardPort={0}&isMain=true（port 为主画面的端口号 值）

*3. 更换主副画面：RecordCmd=ChangePicInPic&isMain=true

*4. 设置小画面的大小和位置：  RecordCmd=SetPositionAndSize&Pos={0},{1}&Size={2},{3}

&CurrPos=LU/LD/RU/RD&isMain=false

*5.关闭小画面：RecordCmd=ClosePicInPic&isMain=false

26）获取mac地址：MetaInfoCmd=GetMac

27）获取ip地址：BroadCastCmd=GetDeviceIP

28）获取磁盘空间状态值：RecordCmd=QueryDiskStatus

29）获取录像时间值：RecordCmd=QueryRecordTime

30）获取系统时间：RecordCmd=QuerySystemTime

31）*保存云台传回的文件信息：

MetaInfoCmd=SaveFile&FileName=UPLOADMETA&FileContent={0}

32）*保存metainfo信息

MetaInfoCmd=SetMetaInfo&Guid={0}&LoginName={1}&CourseName={2}&Tea   cher={3}&Subject={4}&Grade={5}

33）*字幕开启：     RecordCmd=SetCaption& Content={0}&Font={1}&FontColor={2}&BackCol or={3}&FontOpacity={4}&OutlineColor={5}&isMain=true

34）*字幕关闭：RecordCmd=CaptionClose&isMain=true

35）*台标开启：     RecordCmd=SetLogo& PicPath={0}&PicPos={1}&PicSize={2}&PicOpacity ={3}&LogoNum={4}

36）*台标关闭：RecordCmd=LogoClose&LogoNum={0}&isMain=true

37）*手动切换：RecordCmd=ManualTrace

38）*自动切换：RecordCmd= AutoTrace

39）*鼠标点击跟踪：RecordCmd=MouseTrace&Num={0}&X={1}&Y={2}

40）*变倍选择：RecordCmd=Zoom&Num={0}&ZoomNum={1}

41） *图片作为片尾：

   设置：RecordCmd=OpenPicFile&FilePath={0}&Key=3100&StartTime=0

   开始：RecordCmd=PlayPicFile&Key=3101

   停止：RecordCmd=StopPicFile&Key=3101

42）*视频作为片尾：

   设置：RecordCmd=OpenVideoFile&FilePath={0}&Key=3100&StartTime=0

   开始：RecordCmd=PlayVideoFile&Key=3101

   停止：RecordCmd=ChangeCard&CardPort=3101&isMain=true
